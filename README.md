# AI Learning Platform (Next.js)

This repository is refactored from a React + Vite portfolio into a **Next.js App Router AI/ML education platform**.

## Learning Focus

Structured path:

**Python → Mathematics → Machine Learning → Deep Learning → AI Systems**

- `/` Home

- Next.js (App Router)
- React
- Tailwind CSS v4

## Development

```bash
npm install
```
